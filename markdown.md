# Proyecto: Calculadora de Funciones Básicas

## Descripción
Este repositorio contiene una implementación en Python para realizar operaciones matemáticas simples como suma, resta y multiplicación.

## Archivos del Proyecto
* **main.py**: Archivo principal que integra las llamadas a las funciones.
* **funciones.py**: Contiene la lógica de las 3 funciones básicas.
* **.gitignore**: Archivo para ignorar carpetas generadas como `__pycache__`.

## Metodología de Uso
Para ejecutar el proyecto, asegúrate de tener Python instalado y corre el archivo principal:
`python main.py`

## Historial de Trabajo
| Acción | Descripción |
| :--- | :--- |
| Registro inicial | Se agregan 3 funciones básicas en funciones.py |
| Integración | Se integra main.py con llamadas a funciones |
| Limpieza | Se agrega archivo .gitignore para ignorar archivos generados |
